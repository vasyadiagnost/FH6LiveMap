# FH6 Live Map v0.9.22 — Mobile Dock Label Alignment

## Что изменилось

- В мобильной нижней панели выровнены подписи кнопок по единой горизонтальной оси.
- Для иконок нижнего меню задана фиксированная область, чтобы разные символы и SVG не сдвигали подписи вверх/вниз.
- Для кнопок нижнего меню задана аккуратная двухстрочная сетка: иконка сверху, подпись снизу.
