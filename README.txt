FH6 Live Map v0.9.15 compact nearest POI

Start here:
  README.md

Quick run:
  1. Extract the ZIP.
  2. In FH6: Settings -> HUD and Gameplay -> Data Out On.
  3. Set Data Out IP Address to 127.0.0.1 and Data Out Port to 5700.
  4. Double-click run_live_map.bat.
  5. Open http://127.0.0.1:8766 if the browser does not open automatically.

Historical version notes are in CHANGELOG.md.
