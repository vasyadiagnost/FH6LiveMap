# FH6 Live Map v0.9.25 — Meme Layer Freefall Jump Detector

## Что изменилось

- Переработан `jump_takeoff`: теперь это не простой детектор подъёма по высоте, а подтверждённый детектор полёта/свободного падения.
- Событие `jump_takeoff` сохраняет то же имя и ту же папку `data/meme_layer/samples/jump_takeoff/`, чтобы пользовательские сэмплы не переносить.
- Новая логика ждёт недавний подъём/взлёт, затем около секунды падения с отрицательным вертикальным ускорением.
- Обычный подъём по эстакаде или дороге вверх больше не должен считаться прыжком.
- Конфиг мемного слоя автоматически мигрирует на `config_schema_version = 3`.

## Existing samples

Пользовательские сэмплы остаются в тех же папках. Ничего переносить не нужно.
